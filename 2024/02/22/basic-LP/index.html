【学习笔记】线性规划

关于怎么把线性规划格式写得好看点

定义

标准型

我们称一个问题是线性规划，当且仅当它可以被写成如下形式：

maximize⁡∑j=1ncjxjs.t.∑j=1nai,jxj≤bj,∀i=1,2,…,mxj≥0,∀j=1,2,…,n\begin{aligned} \operatorname{maximize}\quad &\sum_{j=1}^{n}c_jx_j\\ \mathrm{s.t.}\quad &\sum_{j=1}^{n}a_{i,j}x_j\le b_j,&\quad& \forall i=1,2,\ldots, m\\ &x_j\ge 0, &\quad& \forall j=1,2,\ldots ,n \end{aligned} maximizes.t.​j=1∑n​cj​xj​j=1∑n​ai,j​xj​≤bj​,xj​≥0,​​∀i=1,2,…,m∀j=1,2,…,n​

这被称作线性规划的标准型。
显然如果我们把 nnn 个数 cjc_jcj​ 和 bjb_jbj​ 分别看作一条行向量，m×nm\times nm×n 个数 aaa 看作矩阵，上述也可以写作矩阵形式，若我们定义 a≤b\boldsymbol a\le \boldsymbol ba≤b 当且仅当 ∀i,xi≤yi\forall i, \boldsymbol x_i\le \boldsymbol y_i∀i,xi​≤yi​，则原式写作：

maximize⁡cTxs.t.Ax≤bx≥0\begin{aligned} \operatorname{maximize}\quad &\boldsymbol c^T\boldsymbol x\\ \mathrm{s.t.}\quad &A\boldsymbol x\le \boldsymbol b\\ &\boldsymbol x\ge 0 \end{aligned} maximizes.t.​cTxAx≤bx≥0​

松弛型

为了更好地处理负数，我们希望将上述的 ≤\le≤ 全部变为 ===，为此，可以使用松弛型：

maximize⁡∑j=1ncjxjs.t.xi+n=bj−∑j=1nai,jxj,∀i=1,2,…,mxj≥0,∀j=1,2,…,n+m\begin{aligned} \operatorname{maximize}\quad &\sum_{j=1}^{n}c_jx_j\\ \mathrm{s.t.}\quad &x_{i+n}=b_j-\sum_{j=1}^{n}a_{i,j}x_j,&\quad& \forall i=1,2,\ldots, m\\ &x_j\ge 0, &\quad& \forall j=1,2,\ldots ,n+m \end{aligned} maximizes.t.​j=1∑n​cj​xj​xi+n​=bj​−j=1∑n​ai,j​xj​,xj​≥0,​​∀i=1,2,…,m∀j=1,2,…,n+m​

这实际上就是把多的部分“寄存”到了一个新的 xxx 里。
不难发现这是一个类似半平面交的半 nnn 维空间交，所以解空间一定是凸的。爬山类的贪心算法不会在某个点终止。

典例数则

最大流

我们人为增加一条边 t→st\rightarrow st→s，容量为 inf\mathrm{inf}inf，从而流量守恒对每个节点均成立。令 f(u,v)f(u,v)f(u,v) 为 u→vu\rightarrow vu→v 的容量，c(u,v)c(u,v)c(u,v) 为 u→vu\rightarrow vu→v 的上界。则：

maximize⁡f(t,s)s.t.f(u,v)≤c(u,v),∀(u,v)∈E∑vf(u,v)=∑vf(v,u),∀u∈Vf(u,v)≥0,∀(u,v)∈E∪{(t,s)}\begin{aligned} \operatorname{maximize}\quad &f(t,s)\\ \mathrm{s.t.}\quad &f(u,v)\le c(u,v),&\quad& \forall (u,v)\in E\\ &\sum_{v}f(u,v)=\sum_{v}f(v,u), &\quad& \forall u\in V\\ &f(u,v)\ge 0, &\quad& \forall (u,v)\in E\cup \{(t,s)\} \end{aligned} maximizes.t.​f(t,s)f(u,v)≤c(u,v),v∑​f(u,v)=v∑​f(v,u),f(u,v)≥0,​​∀(u,v)∈E∀u∈V∀(u,v)∈E∪{(t,s)}​

把每条边的流量看成一个未知数 xxx，就是经典的线性规划了。

最小费用流

注意，此处不必最大化流量。
设每条边的代价为 w(u,v)w(u,v)w(u,v) 而 w(t,s)=0w(t,s)=0w(t,s)=0，那么，不难得到：

minimize⁡∑(u,v)∈Ef(u,v)⋅w(u,v)s.t.f(u,v)≤c(u,v),∀(u,v)∈E∑vf(u,v)=∑vf(v,u),∀u∈Vf(u,v)≥0,∀(u,v)∈E∪{(t,s)}\begin{aligned} \operatorname{minimize}\quad &\sum_{(u,v)\in E}f(u,v)\cdot w(u,v)\\ \mathrm{s.t.}\quad &f(u,v)\le c(u,v),&\quad& \forall (u,v)\in E\\ &\sum_{v}f(u,v)=\sum_{v}f(v,u), &\quad& \forall u\in V\\ &f(u,v)\ge 0, &\quad& \forall (u,v)\in E\cup \{(t,s)\} \end{aligned} minimizes.t.​(u,v)∈E∑​f(u,v)⋅w(u,v)f(u,v)≤c(u,v),v∑​f(u,v)=v∑​f(v,u),f(u,v)≥0,​​∀(u,v)∈E∀u∈V∀(u,v)∈E∪{(t,s)}​

只是这个 minimize⁡\operatorname {minimize}minimize 看起来不像是上述提到的形式？这一点并没有太大的关系，在下一节中，就会阐述这其中的道理。

像上面，一样我们可以把系数写成 边数×点数边数\times 点数边数×点数 的矩阵。再乘上一条 1×边数1\times 边数1×边数 的未知数行向量并最终得到一条 000 向量。
请注意，在满足流量守恒时，这个矩阵每一行有且只有两个数非 000 且分别为 +1+1+1 和 −1-1−1（作为出边的贡献和作为入边的贡献）。这一点性质非常重要。

线性规划对偶

对偶，简单来说就是把约束和目标函数地位对换。
我们定义线性规划

maximize⁡∑j=1ncjxjs.t.∑j=1nai,jxj≤bi,∀i=1,2,…,mxj≥0,∀j=1,2,…,n\begin{aligned} \operatorname{maximize}\quad &\sum_{j=1}^{n}c_jx_j\\ \mathrm{s.t.}\quad &\sum_{j=1}^{n}a_{i,j}x_j\le b_i,&\quad& \forall i=1,2,\ldots, m\\ &x_j\ge 0, &\quad& \forall j=1,2,\ldots ,n \end{aligned} maximizes.t.​j=1∑n​cj​xj​j=1∑n​ai,j​xj​≤bi​,xj​≥0,​​∀i=1,2,…,m∀j=1,2,…,n​

的对偶问题是：

minimize⁡∑i=1mbiyis.t.∑i=1mai,jyi≥cj,∀j=1,2,…,nyj≥0,∀i=1,2,…,m\begin{aligned} \operatorname{minimize}\quad &\sum_{i=1}^{m}b_iy_i\\ \mathrm{s.t.}\quad &\sum_{i=1}^{m}a_{i,j}y_i\ge c_j,&\quad& \forall j=1,2,\ldots, n\\ &y_j\ge 0, &\quad& \forall i=1,2,\ldots ,m \end{aligned} minimizes.t.​i=1∑m​bi​yi​i=1∑m​ai,j​yi​≥cj​,yj​≥0,​​∀j=1,2,…,n∀i=1,2,…,m​

或者，更清楚地：

maximize⁡cTxs.t.Ax≤bx≥0\begin{aligned} \operatorname{maximize}\quad &\boldsymbol c^T\boldsymbol x\\ \mathrm{s.t.}\quad &A\boldsymbol x\le \boldsymbol b\\ &\boldsymbol x\ge 0 \end{aligned} maximizes.t.​cTxAx≤bx≥0​

与

minimize⁡bTys.t.ATy≥cy≥0\begin{aligned} \operatorname{minimize}\quad &\boldsymbol b^T\boldsymbol y\\ \mathrm{s.t.}\quad &A^T\boldsymbol y\ge \boldsymbol c\\ &\boldsymbol y\ge 0 \end{aligned} minimizes.t.​bTyATy≥cy≥0​

互为对偶。

接下来，我们不加证明地给出如下结论：

  • 弱对偶定理：

    对于任意一组原问题（maximize⁡\operatorname{maximize}maximize）的解，与其对偶问题（minimize⁡\operatorname{minimize}minimize）的任意一组解，均有

    若 x\boldsymbol xx 与 y\boldsymbol yy 分别为两个问题的一组最优解，那么：

    若 x\boldsymbol xx 与 y\boldsymbol yy 分别为原问题与对偶问题的一组解，则它们都是最优解，当且仅当以下条件同时满足：

    1. ∀1≤i≤m,yi=0∨∑j=1nai,jxj=bi\forall 1\le i\le m,y_i=0\lor \sum_{j=1}^{n}a_{i,j}x_j=b_i∀1≤i≤m,yi​=0∨∑j=1n​ai,j​xj​=bi​
    2. ∀1≤j≤n,xj=0∨∑i=1mai,jyi=cj\forall 1\le j\le n,x_j=0\lor \sum_{i=1}^{m}a_{i,j}y_i=c_j∀1≤j≤n,xj​=0∨∑i=1m​ai,j​yi​=cj​
      更形象地说，若 x\boldsymbol xx 是原问题的一组最优解，则要么第 iii 条限制是紧的，要么对偶变量 yiy_iyi​ 是紧的（等于 000）。
      这条定理揭示了原问题的最优解方案与对偶问题方案的最优解的联系。
