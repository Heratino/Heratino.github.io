CF1117E Decypher the String

CF1117E Decypher the String

题解

当 len≤26len\le 26len≤26 的时候问题是非常简单的。我们直接询问 abcd...z，考虑第 iii 个字符到的位置 jjj，然后令 ansi=sjans_i=s_jansi​=sj​ 就行了。

那么 len=∣Σ∣len=|\Sigma|len=∣Σ∣ 的情况显然就是这样，但是显然我们 ∣Σ∣=26|\Sigma|=26∣Σ∣=26 没法整这种东西，怎么办呢！其实这个告诉我们我们的关键就是要知道每个位置最后被换到了哪里。

如果不在乎询问次数，我们可以每次询问 aaabaaaa 这种，然后考虑最后 b 的位置，我们就知道了 b 最后被换到了什么位置。

用 O(n)\mathcal O(n)O(n) 次询问就能问出来最后的答案了！但是这个东西显然有点过于笨蛋了，我们每次就用了两种字符。

考虑把两种做法结合起来，先想 len≤262len\le 26^2len≤262 怎么做，这个时候我们可以这样询问

  1. abcd...xyzabcd...xyz...，用 abcd...xyz 循环。
  2. aaa...a(26)bbb...b(26)...zzz...z(26) 用每段长度为 262626 的字符连续段循环。

那么换位的时候，假设第 iii 个位置在第一个串里面被换成了 cic_ici​，而在第二个串里面没有变，说明一定是和自己这段长度为 262626 的同色段里，第一段中的 cic_ici​ 交换的位置。

如果第 iii 个位置在第二个串里面被换成了 pip_ipi​，第一个串里变成了 cic_ici​，说明是和 pip_ipi​ 那一段里面的 cic_ici​ 做了交换。

所以我们就确定了每个位置被换到的位置。

所以我们就会做 len≤263len\le 26^3len≤263 了，只要再询问一个

  1. aaa...a(26*26)bbb...b(26*26)...zzz...z(26*26)

确定它在第三层的位置即可。
