CF1264D2 Beautiful Bracket Sequence

简要题意

给你一个长度为 nnn 的，只有 ?，(，) 的字符串 sss，定义一个括号序列的深度是在删除若干个字符后，形如 (((()))) 的字符串嵌套的最大层数。

求将所有 ? 变为 ( 或 ) 的所有方案的深度和。

题解

先想 D1 的 n≤2×103n\le 2\times 10^3n≤2×103 怎么做

我们首先要会算一个给定序列的深度，每次令没匹配的最右边的括号与最左边的括号匹配即可。

两个都满足的时候要减去 fl+1,r−1f_{l+1,r-1}fl+1,r−1​，因为重复统计了。

当 sls_lsl​ 为 ( 或 ?，srs_rsr​ 为 ) 或 ? 时，根据上面的结论，左边的 ( 一定不会和 [l+1,r−1][l+1,r-1][l+1,r−1] 的 ) 匹配。

所以我们对于 fl+1,r−1f_{l+1,r-1}fl+1,r−1​ 所代表的每种情况，fl,rf_{l,r}fl,r​ 加上其深度 +1+1+1。考虑这里会有几种情况，实际上每个 ? 是相互独立的，所以共有 222 的 ? 个数次方种情况，这个记为 ttt，则 fl,r=fl+1,r−1+tf_{l,r}=f_{l+1,r-1}+tfl,r​=fl+1,r−1​+t

这样就是 O(n2)O(n^2)O(n2) 的 DP。

不难发现这东西没啥前途，考虑换个做法。

考虑上面说的，每次让没匹配的最右边的括号与最左边的括号匹配。

这实际上可以变成，我们枚举一个划分点 iii，计算左边的左括号数量 aia_iai​，右边的有括号数量 bib_ibi​。

所以我们要求的就是 max⁡(min⁡(ai,bi))\max(\min(a_i,b_i))max(min(ai​,bi​))，根据 「联合省选 2020 A | B」冰火战士 这个题的经验，aia_iai​ 单调递增，bib_ibi​ 单调递减，极值肯定在两条线交点附近取到。而这道题，交点处一定是极值。
